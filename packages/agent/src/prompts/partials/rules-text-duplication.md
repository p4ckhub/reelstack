## TEXT-CARD STYLE RULES

- text-card backgrounds: use DARK gradients (e.g. "linear-gradient(135deg, #0F172A, #1E293B)"), NOT plain solid colors
- Keep text-card content minimal: short headline (2-4 words) + optional subtitle. No emoji in headlines.
- NEVER put the same text in both headline AND subtitle. headline = main point, subtitle = context that adds NEW information not in headline.

(For text overlap rules with captions / lower-thirds / text-emphasis, see the "NO TEXT REDUNDANCY WITH CAPTIONS" section earlier in this prompt.)
