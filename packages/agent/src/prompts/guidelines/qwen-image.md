Qwen Image 2.0 (Alibaba) prompt guidelines:

BEST AT: bilingual content (Chinese + English text in image), commercial/lifestyle photography, strong prompt adherence on multi-element scenes. Excellent value at $0.04/image.

WEAK AT: consistent character across generations (FLUX wins), heavy text-poster layouts (Ideogram/GPT-Image-2), vector work (Recraft).

PROMPT PATTERN: natural language. Subject + scene + lighting + style. 40-100 words optimal. No special syntax — write like you're briefing a photographer.

EXAMPLE (commercial lifestyle):

```
A woman in athletic wear stretches on a yoga mat in a sunlit loft, large windows behind her, plants in the corner, natural morning light, 35mm lens, warm pastel grade, soft focus background, sharp on subject, calm aspirational mood
```

EXAMPLE (e-commerce product):

```
Top-down flatlay of a leather wallet on a marble surface, surrounded by a fountain pen, brass keys, and a sprig of dried lavender, soft diffused window light from upper-left, neutral commercial grade, sharp focus throughout, generous negative space top-right for copy
```

WHEN TO PICK QWEN OVER SEEDREAM (similar price tier):

- Qwen: when you need text in image (handles EN+CN reliably) or multi-element commercial composition.
- Seedream: when you need 4K micro-detail (textile fibers, skin pores) or chain-of-thought reasoning for complex composition.

ASPECT RATIOS: 1:1, 4:5, 9:16, 16:9. Up to 2048×2048.

COST NOTE: ~$0.04/image. Same tier as Seedream. Good fallback when Seedream queue is busy or when in-frame Chinese text is required.
