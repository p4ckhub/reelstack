Seedream 4.5 / 5.0 (ByteDance) prompt guidelines:

BEST AT: photorealistic 4K with micro-detail preservation (textile fibers in clothing, pore-level skin, filament-level grass), product shots, lifestyle photos at scale. Generates in 2-3s. Seedream 5.0 Lite adds chain-of-thought visual reasoning — it can plan composition before drawing.

WEAK AT: consistent character across multiple generations (FLUX wins), heavy in-frame text layouts (use Ideogram/GPT-Image-2), vector/icon (use Recraft).

PROMPT PATTERN: natural language. No special syntax. Describe like you're directing a photographer.

- Subject + action + environment, lens/framing, lighting, mood. 50-100 words optimal.
- Bilingual: Chinese + English. For PL/EN content, write in English — model is trained predominantly on EN captions.

EXAMPLE (lifestyle):

```
A young barista pulls an espresso shot at a sunlit cafe counter, steam rising from the portafilter, focused expression, 50mm natural perspective, soft morning window light from camera-left, warm wood interior, muted desaturated grade, sharp focus on hands and machine
```

EXAMPLE (product, 4K micro-detail):

```
Macro shot of a luxury watch on a slate stone surface, brushed steel case with visible micro-grain, leather strap with stitching detail, single hard key light from above-right with soft fill, dark dramatic background, 90mm macro lens, f/8 sharp throughout, neutral commercial grade
```

WHEN TO PICK SEEDREAM:

- vs FLUX: Seedream when you need budget-friendly volume + 4K detail. FLUX when character must stay consistent across shots.
- vs NanoBanana 2: Seedream when you don't need conversational edits. NanoBanana when iterating on one image.

ASPECT RATIOS: 1:1, 4:5, 9:16, 16:9. Native 4MP (2048×2048).

COST NOTE: ~$0.04/image at native 4MP. Among the cheapest 4K-capable models. Good for bulk reel B-roll where each shot needs polish.
